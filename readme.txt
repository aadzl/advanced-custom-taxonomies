Customise WordPress with powerful, professional and intuitive taxonomies.

== Description ==

Advanced Custom Taxonomies is the perfect solution for any WordPress website which needs custom taxonomies.

== Filters ==

**actax/settings/show_admin** - Determines whether or not the admin menu is shown.

**actax/settings/capability** - Determines capability is needed to manage custom taxonomies.

**actax/settings/save_json** - The path to save taxonomies locally.

== Actions ==

**actax/init** - Fires before actax has started loading.
